# fantastic-node
meshtastic node designed to be solar powered and small/unnoticeable enough to be planted in urban spaces.

# Project BOM

Here you will find all the parts used and their prices as link to them. From here you can estimate the cost of building one node.

One solar pannel should be at least 0.5w to ensure the node will stay powered with bad weather. Optimally should be 5v or 6v solar module. One pannel cost up to $5

Lithium ion cell, that can be pouch or cylindrical. Up to $3

PCB cost $2+shipping so about $10-15 total
